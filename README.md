# Focus Check Screeners

Free, open-source mental health self-reflection tools.

## Available Screeners

- **Steady Check** — Anxiety Self-Reflection Screener (`screens/anxiety-steady-check.html`)
- **ADHD Symptom Check-In** (`screens/adhd-screening.html`)
- **OCD Symptom Check-In** (`screens/ocd-screening.html`)

## Usage

All screeners are standalone HTML files. Open any file in a web browser to use.

**Disclaimer:** These tools are for self-reflection only and are not diagnostic instruments. Always consult a qualified mental health professional for diagnosis and treatment.
